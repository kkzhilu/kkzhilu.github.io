阿里云Linux安装mysql，修改密码，远程连接

博文：https://www.cnblogs.com/dianqijiaodengdai/p/7731562.html

1.安装yum包

进入mysql官网 https://dev.mysql.com/downloads/file/?id=470281

获取mysql57-community-release-el7-11.noarch.rpm 下载地址

在root下输入指令：wgethttps://dev.mysql.com/get/mysql57-community-release-el7-11.noarch.rpm

2.rpm源下载完成后，输入指令

yum install mysql-server
安装依赖包过程一直输入y，至安装完毕

3.相关命令

输入systemctl start mysqld 启动mysql
输入systemctl status mysqld 查询mysql状态

4.获取初始密码

more /var/log/mysqld.log
找到 "temporary password"所在行，获取初始密码

5.重置密码 - yum安装密码有一定的限制，因此先要修改密码规则:

首先要修改规则链接地址：https://www.cnblogs.com/ivictor/p/5142809.html
1.set global validate_password_policy=0;
2.set global validate_password_length=1;
// 3.set global validate_password_mixed_case_count=2;

修改密码为：123456
命令:alter user 'root'@'localhost' identified by 'root'

6.检查端口是否开启 - 主要是阿里云安全组的端口，查看是否开启即可

7.开启远程链接，反思：注意看到代码先分析

grant all privileges on *.* to 'root'@'%' identified by '123456';
flush privileges;

8.本地navicat测试连接

额外

提示：Failed to start mysqld.service: Unit not found

解决方案：https://my.oschina.net/iyinghui/blog/2246986
