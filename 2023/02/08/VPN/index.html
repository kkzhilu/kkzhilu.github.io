同一局域网如何连接

  1. 打开Clash的Allow Lan

  2. 通过 ipconig 获取使用VPN电脑的IP地址，同时查看 Clash 打开的端口号

  3. 手机或其它设备先连接上同一局域网，进入手机 WiFi 的详细设置界面，把选项 代理 从 无 改成 手动 ，选项 主机名 设置为 IP地址，选项 代理服务器端口 改为 端口，确认即可

  4. 其他电脑可以使用 SwitchyOmega 直接连接目标电脑的 IP + 端口即可
