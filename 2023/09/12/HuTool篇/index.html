HuTool

Hutool是一个小而全的Java工具类库，通过静态方法封装，降低相关API的学习成本，提高工作效率，使Java拥有函数式语言般的优雅，让Java语言也可以“甜甜的”。

Hutool中的工具方法来自于每个用户的精雕细琢，它涵盖了Java开发底层代码中的方方面面，它既是大型项目开发中解决小问题的利器，也是小型项目中的效率担当；

Hutool是项目中“util”包友好的替代，它节省了开发人员对项目中公用类和公用工具方法的封装时间，使开发专注于业务，同时可以最大限度的避免封装不完善带来的bug

依赖

<dependency>
    <groupId>cn.hutool</groupId>
    <artifactId>hutool-all</artifactId>
    <version>5.3.8</version>
</dependency>

public class App {

    /***
     * 基于HuTool 多参考已有的工具类库，提高稳定性及效率
     */
    public static void main(String[] args) {

        // 普通Get请求
        Map<String, Object> normalGetMap = new HashMap<>();
        normalGetMap.put("type", "more");
        normalGetMap.put("category", "home");
        normalGetMap.put("shown_offset", "1593314417949658");

        String normalGet = HttpUtil.get("https://www.csdn.net/api/articles", normalGetMap);
        System.out.println(normalGet);

        // 简单Get请求
        System.out.println(HttpUtil.get("https://www.csdn.net/api/articles?type=more&category=home&shown_offset=1593314417949658"));

        // Header型 Get请求
        String complexGet = HttpRequest.get("https://kunpeng.csdn.net/ad/474")
                .header("k1", "v1").header("k2", "v2").form(normalGetMap).execute().body();

        // 注意最终可获取 HttpResponse 对象
        HttpResponse httpResponse = HttpRequest.get("https://kunpeng.csdn.net/ad/474")
                .header("k1", "v1").header("k2", "v2").form(normalGetMap).execute();


        /**************************************** 分割线 ***********************************/


        // 普通Post请求
        Map<String, Object> normalPostMap = new HashMap<>();
        normalPostMap.put("coupon", true);

        String resultPost = HttpUtil.post("https://msg.csdn.net/v1/web/message/view/unread", "{\"coupon\":true}");
        String resultPostMap = HttpUtil.post("https://msg.csdn.net/v1/web/message/view/unread", normalPostMap);
        System.out.println(resultPost);
        System.out.println(resultPostMap);

        // Header型 Post请求
        String response = HttpRequest.post("https://msg.csdn.net/v1/web/message/view/unread")
                .header("cookie", "xxx").body("{\"coupon\":true}").execute().body();
        System.out.println(response);
    }
}
